# Выборка по шаблону

## LIKE 

сравнивает строку с шаблоном, который может включать два специальных символа

##### % — соответствует любому числу символов, включая 0
##### _ — соответствует ровно одному любому символу

```
SELECT * FROM <table_name>
WHERE <column> LIKE <pattern>;
```

### Можно применять функции строк

```
WHERE LOWER(name) LIKE 'a%'
```

### SQLite

> [!WARNING]
> в SQLite LIKE регистронезависимый

надо применять:

```
PRAGMA case_sensitive_like = ON;
```
