# Агрегация данных

## COUNT

определяет количество записей в таблице или выборке

```
COUNT(*)
```

COUNT считает только строки, где некоторое выражение expr не равно NULL:
```
COUNT(expr)
```

## SUM

```
SUM(column)
```
> если в столбце, на котором применяется SUM, встречаются значения NULL, они игнорируются

## MAX и MIN

MAX и MIN возвращают крайние значения столбца

```
MAX(column)
MIN(column)
```

## AVG

вычисляет среднее арифметическое значения столбца

```
AVG(column)
```

## ROUND

округление до ближайшего целого
```
ROUND(value)
```

округление до digits знаков после запятой
```
ROUND(value, digits)
```
