# Строковые функции

## LENGTH

определить длину строки

```
LENGTH(name)
```

## UPPER и LOWER

изменить регистр букв

```
UPPER(name)
```

## SUBSTR

позволяет извлечь часть строки

start - номер начального символа
length - количество символов (необязательная опция; если не указано, то до конца строки)

```
SUBSTR(text, start, length)
```

## INSTR

Функция INSTR возвращает позицию первого вхождения подстроки в строке

```
INTSTR(name, '.')
```

## TRIM

TRIM() удаляет пробелы с обеих сторон

LTRIM() и RTRIM() удаляют пробелы слева и справа соответственно.

```
TRIP(name)
```

> TRIP('    сообщение    ') -> 'сообщение'

## REPLACE(text, old, new)

```
SELECT REPLACE('2025-11-13', '-', '/');
```
