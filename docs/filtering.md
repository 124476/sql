# Фильтрацией данных

## WHERE

позволяет отфильтровать данные

```
SELECT столбцы
FROM таблица
WHERE условие;
```

### Условия

- оператор сравнения (>, <, =, !=, >=, <=)
- логические связки (AND, OR, NOT)
- фильтрацию по диапазонам (BETWEEN)
- фильтрацию по множеству значений (IN)
- проверки с текстовыми функциями
- арифметические выражения

### Работа с датами

```
SELECT * FROM events
WHERE date < '2026-01-01';
```

### Работа со строками

SQL сравнивает строки лексикографически, то есть по алфавиту

### Работа с NULL

```
price IS NULL
price IS NOT NULL
```

> [!WARNING]
> Нельзя писать price != NULL

### Приоритет операторов

NOT -> AND -> OR

### BETWEEN 

проверяет, попадает ли значение в диапазон от минимального значения до максимального

```
value BETWEEN min AND max    
```

> [!NOTE]
> тоже самое, что 
> ```value >= min AND value <= max```

> [!WARNING]
> BETWEEN включает оба края диапазона

### IN

проверяет, принадлежит ли значение столбца одному из элементов перечисления

```
value IN (value1, value2, value3, ...)
```

```
SELECT * FROM products
WHERE category IN ('Category1', 'Category2', 'Category3');
```
